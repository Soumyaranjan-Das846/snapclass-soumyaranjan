## 📖 Overview

SnapClass is a modern AI-powered attendance management platform designed to automate and simplify attendance tracking in educational institutions.

The system combines Face Recognition, Voice Authentication, QR/Join Code Enrollment, and Real-Time Analytics to provide a secure and efficient attendance experience for both teachers and students.

---

## ✨ Features

### 👨‍🎓 Student Features

- Secure Login & Authentication
- Join Subjects using Subject Code
- Quick Enrollment via Shared Links
- View Attendance Statistics
- Track Attendance Records
- Manage Enrolled Subjects
- One-click Unenroll Support

### 👨‍🏫 Teacher Features

- Create & Manage Subjects
- Generate Subject Join Codes
- Share Enrollment Links
- Face Recognition Attendance
- Voice Authentication Attendance
- Attendance Analytics Dashboard
- View Historical Attendance Records

### 🤖 AI Features

- Face Recognition Pipeline
- Voice Authentication Pipeline
- Attendance Verification
- Automated Attendance Processing

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Soumyaranjan-Das846/snapclass-soumyaranjan.git
```

Move into project directory:

```bash
cd snapclass-soumyaranjan
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
