## 🚀 Overview

SnapClass is an AI-powered attendance management platform designed to eliminate manual attendance processes in classrooms.

The system enables teachers to create courses, manage students, track attendance records, and automate classroom workflows through intelligent technologies.

---

## ✨ Features

### 👨‍🏫 Teacher Module
- Secure authentication
- Course creation & management
- Student enrollment tracking
- Attendance monitoring dashboard
- Attendance report generation

### 👨‍🎓 Student Module
- Quick enrollment
- Course registration
- Attendance visibility
- Streamlined classroom access

- Face Recognition Attendance

## ⚙️ Installation

### Clone Repository

```bash

```

```bash

```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
